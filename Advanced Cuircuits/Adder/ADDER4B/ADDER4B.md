# 4-Bit Adder 详细说明

## 描述：

ADDER4B 是 FULLADDER 的 4 位总线版本。因此，它有两条 4 位输入总线和一条 4 位输出总线，同时保留了进位和出位。  
查看手册中的附录，了解数字的二进制表示法和二进制算术。

## 接口规范：

Inputs: carryIn, in1[4], in2[4];
Outputs: out[4], carryOut;

## 图形化表示：

```
-carryIn----\
          __|____
-in1[4]--|       |
         | ADD4B |--out[4]-
-in2[4]--|_______|
              |
              \--carryOut--

```
