# 4-Bit Multiplexer 详细说明

## 描述：

MUX4B 元件是 MUX 门的总线版本。它将其中一条输入总线的全部数据传输到输出总线，但在其他方面行为相同。

## 接口规范：

输入: in1[4], in2[4], sel;  
输出: out[4];

## 图形化表示：

```
          ____
-in1[4]--|    \
         | MUX |--out[4]-
-in2[4]--|____/
           |
-sel-------/

```

## 真值表：

| sel | in1  | in2  | out  |
| :-: | :--: | :--: | :--: |
|  0  | 1010 | 0101 | 1010 |
|  1  | 1010 | 0101 | 0101 |
