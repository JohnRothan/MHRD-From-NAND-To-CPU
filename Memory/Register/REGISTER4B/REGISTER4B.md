# 4-Bit Register 详细说明

## 描述：

REGISTER4B 是 REGISTER 元件的总线版本。

## 接口规范：

输入: in[4], load;
输出: out[4];

## 图形化表示：

```
         _______
        |       |
-in[4]--| REG4B |--out[4]-
        |_______|
            |
-----load---/


```

## 真值表：

|  in  | load | out  |
| :--: | :--: | :--: |
| 0101 |  0   | 0000 |
| 1010 |  0   | 0000 |
| 0101 |  1   | 0000 |
| 1010 |  0   | 0101 |
| 1111 |  1   | 0101 |
| 0000 |  0   | 1111 |
