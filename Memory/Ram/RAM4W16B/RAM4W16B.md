# 4-Register 16-Bit RAM 详细说明

## 描述：

RAM4W16B 是一个随机存取存储器，具有 4 个可寻址寄存器和 16 位总线宽度。除了常规寄存器外，RAM4W16B 还可以保存和检索 4 个不同的值，这些值可以通过地址输入总线进行寻址。

## 接口规范：

输入: in[16], load, address[2];  
输出: out[16];

## 图形化表示：

```
          __________
         |          |
-in[16]--| RAM4W16B |--out[16]-
         |__________|
            | |
-----load---/ |
-address[2]---/

```
