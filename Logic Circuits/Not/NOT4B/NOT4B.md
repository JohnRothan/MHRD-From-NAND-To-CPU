# 4-Bit NOT 详细说明

## 描述：

NOT4B元件 是 NOT门 的总线版本。它将 Not-operation 应用于总线中的每一个位。

## 接口规范：

输入：in[4];  
输出：out[4];

## 图形化表示：

```
         _ _ _ _ _ _
        ┆           ┆
-in[4]--┆  NOT4B    ┆--out[4]-
        ┆_ _ _ _ _ _┆

```

## 真值表：

|  in  | out  |
| :--: | :--: |
| 0000 | 1111 |
| 1111 | 0000 |
| 0101 | 1010 |
| 1010 | 0101 |
| 0110 | 1001 |
| 1001 | 0110 |
