# 4-Way Or 详细说明

## 描述：

OR4W 元件是 OR 门 的总线版本。如果总线中至少一个位为 1，输出为 1。否则输出为 0。

## 接口规范：

输入：in[4];  
输出：out;

## 图形化表示：

```
          _ _ _ _ _ _
         ┆           ┆
-in[4]-- ┆   OR4W    ┆--out-
         ┆_ _ _ _ _ _┆

```

## 真值表：

|  in  | out |
| :--: | :-: |
| 0000 |  0  |
| 1111 |  1  |
| 0101 |  1  |
| 1010 |  1  |
| 0110 |  1  |
| 1001 |  1  |
